**VillageSQL Server for MySQL** は、MySQL のそのまま差し替え可能な、オープンソースで拡張可能なトラッキングフォークです。

VillageSQL Server は、MySQL の堅牢な基盤を基に、強力な **拡張フレームワーク** を導入しています。このフレームワークにより、開発者は、アップストリームリリースを待ったり、複雑なフォークを維持したりすることなく、新しいデータ型や関数などのまったく新しい機能を追加できます。

このバージョンの VillageSQL は、MySQL 8.4.9 と互換性があります。

**<u>VillageSQL Server バージョン 0.0.4 は開発中です。本番環境での使用は避けてください</u>**
