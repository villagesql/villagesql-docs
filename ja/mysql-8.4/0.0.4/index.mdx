**VillageSQL Server for MySQL** は、MySQL のためのオープンソースで拡張機能をサポートする追跡フォークであり、MySQL のドロップイン置き換えです。

VillageSQL Server は MySQL の堅牢な基盤を活かしつつ、強力な **拡張フレームワーク** を導入しています。このフレームワークにより、開発者はアップストリームリリースを待たず、複雑なフォークを維持することなく、完全に新しい機能（データ型や関数など）を追加できます。

このバージョンの VillageSQL は MySQL 8.4.9 と互換性があります。

**<u> VillageSQL Server バージョン 0.0.4 は開発中です。本番環境での使用はお控えください。</u>**
