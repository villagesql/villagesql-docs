VillageSQLは組み込みの拡張機能を備えており、VillageSQLチームによって構築および維持される公式拡張機能をサポートしています。それらのいずれかを`INSTALL EXTENSION`でインストールできます。

## 組み込み拡張機能

<Warning>
  ソースからビルドする際には、`vsql_complex`および`vsql_simple`のみが含まれます。以下の公式拡張機能は別途のリポジトリで維持されています。
</Warning>

### vsql_complex

**ステータス:** VillageSQLに組み込み
**リポジトリ:** [villagesql-server/villagesql/examples/vsql-complex](https://github.com/villagesql/villagesql-server/tree/main/villagesql/examples/vsql-complex)

電気工学、信号処理、および科学計算向けの完全な算術サポートを備えた複素数データ型。

**機能:**
- 複素数（a + bi）を格納するための`COMPLEX`カラム型
- 文字列リテラル形式: `'(real,imaginary)'`（例: `'(3.0,4.0)'`）
- 算術: 加算、減算、乗算、除算
- ユーティリティ: `complex_real()`, `complex_imag()`, `complex_abs()`, `complex_conjugate()`
- バイナリストレージ: 16バイト（2倍精度浮動小数点数）

**インストール:**
```sql
INSTALL EXTENSION vsql_complex;
```

**例:**
```sql
CREATE TABLE signals (
    id INT PRIMARY KEY,
    impedance COMPLEX,
    frequency_response COMPLEX
);

INSERT INTO signals VALUES
    (1, '(50.0,0.0)', '(0.95,0.31)');

SELECT
    complex_abs(impedance) as magnitude,
    complex_add(impedance, frequency_response) as total
FROM signals;
```

## コア拡張機能

以下の拡張機能は別途のリポジトリで維持されており、個別にビルドおよびインストールする必要があります。各拡張機能には独自のインストール手順およびドキュメントが提供されています。

### vsql_ai

**リポジトリ:** [villagesql/vsql-ai](https://github.com/villagesql/vsql-ai)

AIを活用したSQL関数。SQLクエリ向けのAIプロンプト処理および埋め込み関数を提供します。Anthropic Claude、Google Gemini、OpenAI、およびローカルのOllamaをサポートしています。

### vsql_crypto

**リポジトリ:** [villagesql/vsql-crypto](https://github.com/villagesql/vsql-crypto)

暗号化関数。ハッシュ（MD5、SHA-1、SHA-256、SHA-512など）、HMAC、AES暗号化、パスワードハッシュ、OpenSSLを使用したセキュアなランダムデータ生成をサポートします。

### vsql_cube

**リポジトリ:** [villagesql/vsql-cube](https://github.com/villagesql/vsql-cube)

N次元ボックスおよびポイント型。最大100次元で幾何学的ボックスおよびポイントを格納およびクエリ可能で、コンストラクタ、アクセサ、述語、距離、幾何学関数を提供します。VillageSQL 0.0.4が必要です。

### vsql_http

**リポジトリ:** [villagesql/vsql-http](https://github.com/villagesql/vsql-http)

HTTPクライアント関数。SQLクエリからHTTPリクエスト（GET、POST、PUT、DELETE、PATCH）を実行し、JSONレスポンスを返すほか、URLエンコードおよびデコードをサポートします。

### vsql_network_address

**リポジトリ:** [villagesql/vsql-network-address](https://github.com/villagesql/vsql-network-address)

ネットワークアドレス型。IPv4、IPv6、MACアドレスのデータ型を検証、比較、ネットワーク操作をサポートします。

### vsql_uuid

**リポジトリ:** [villagesql/vsql-uuid](https://github.com/villagesql/vsql-uuid)

UUIDデータ型および関数。SQLでネイティブにUUIDを生成および操作できます（v1、v3、v4、v5、v6、v7）。ネイティブな16バイトUUID型、タイムスタンプの抽出、比較をサポートします。

## コミュニティ拡張機能

コミュニティ拡張機能は、VillageSQL拡張フレームワーク（VEF）を使用してサードパーティによって構築されます。VillageSQLチームによって維持されていません。

### vsql_statistics

**リポジトリ:** [ronaldbradford/vsql-statistics](https://github.com/ronaldbradford/vsql-statistics)
**メンテナー:** [Ronald Bradford](https://github.com/ronaldbradford)

データ分析用の統計集約関数。IQR、四分位数、中央値、Tukeyのフェンスによる外れ値検出 — `STATS_IQR`, `STATS_Q1`, `STATS_Q3`, `STATS_MEDIAN`, `STATS_IQR_LOWER_FENCE`, `STATS_IQR_UPPER_FENCE`。VillageSQL 0.0.4が必要です。

### prometheus_exporter

**リポジトリ:** [ProxySQL/vsql-prometheus-exporter](https://github.com/ProxySQL/vsql-prometheus-exporter)
**メンテナー:** [ProxySQL](https://github.com/ProxySQL)

組み込みPrometheusメトリクスエクスポーター。HTTP `/metrics`エンドポイントでPrometheusの公開フォーマットを使用してMySQLサーバーのメトリクスを公開します — グローバルステータス、グローバル変数、InnoDBメトリクス、レプリケーションステータス。
