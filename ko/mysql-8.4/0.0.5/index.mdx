**VillageSQL Server for MySQL**은 기존 환경에 바로 대체 가능한 오픈소스이자, 확장 기능을 지원하며 업스트림을 추적하는 포크입니다.

VillageSQL Server는 MySQL의 견고한 기반 위에 구축되며 강력한 **확장 프레임워크**를 도입합니다. 이 프레임워크를 통해 개발자는 업스트림 릴리스를 기다리지 않고 복잡한 포크를 유지 관리하지 않아도 새로운 데이터 유형 및 함수와 같은 완전히 새로운 기능을 추가할 수 있습니다.

이 VillageSQL 버전은 MySQL 8.4.10과 호환됩니다.

**<u> VillageSQL Server 버전 0.0.5는 개발 중입니다. 프로덕션 환경에서는 사용하지 마세요.</u>**
