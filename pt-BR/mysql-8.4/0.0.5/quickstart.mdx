Coloque uma instância do VillageSQL Server em execução, conecte-se a ela e experimente o sistema de extensões.

## Passo 1: Instale o VillageSQL

### Opção A: Docker (Recomendado)

Execute o VillageSQL em um contêiner, sem nenhuma instalação no host:

```bash
docker run -d --name vsql -e MYSQL_ALLOW_EMPTY_PASSWORD=yes -p 3306:3306 villagesql/server:stable
```

### Opção B: Script de Shell

Instale o VillageSQL diretamente na sua máquina usando o script de instalação oficial. Ele baixa e configura o binário do servidor para a sua plataforma.

```bash
curl -fsSL https://install.villagesql.com | bash
```

Para inspecionar o script antes de executá-lo: `curl -fsSL https://install.villagesql.com | less`

### Opção C: Compilar a Partir do Código-Fonte

Para desenvolvimento ou compilações personalizadas, siga o [Compilar a Partir do Código-Fonte](/pt-BR/mysql-8.4/0.0.5/source) para compilar a partir do código mais recente.

## Passo 2: Conecte-se ao Servidor

Assim que o servidor estiver em execução, conecte-se usando qualquer cliente MySQL padrão:

```bash
mysql -h 127.0.0.1 -P 3306 -u root -p
```

## Passo 3: Instale Sua Primeira Extensão

O VillageSQL vem com extensões de exemplo que você pode instalar imediatamente.

Instale a extensão `vsql_complex` para adicionar suporte a números complexos:

```sql
INSTALL EXTENSION vsql_complex;
```

Verifique a instalação:

```sql
SELECT * FROM INFORMATION_SCHEMA.EXTENSIONS;
```

Você deverá ver `vsql_complex` listada.

Para mais detalhes, consulte [Instalando Extensões](/pt-BR/mysql-8.4/0.0.5/install).

## Passo 4: Use Tipos de Dados Estendidos

Agora que a extensão está ativa, você pode usar o tipo de dados `COMPLEX` em suas tabelas, assim como os tipos nativos.

```sql
-- Create a database and use it
CREATE DATABASE demo;
USE demo;

-- Create a table using the new COMPLEX type
CREATE TABLE electronic_components (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    impedance COMPLEX
);

-- Insert complex number values (using string literals)
INSERT INTO electronic_components VALUES
    (1, 'Resistor', '(100,0)'),
    (2, 'Inductor', '(0,50)'),
    (3, 'Capacitor', '(25,-75)');

-- Query the data
SELECT * FROM electronic_components;

-- Use extension functions
SELECT
    name,
    impedance,
    complex_abs(impedance) AS magnitude,
    complex_real(impedance) AS resistance,
    complex_imag(impedance) AS reactance
FROM electronic_components;
```

Para desinstalar uma extensão:

```sql
UNINSTALL EXTENSION vsql_complex;
```

## Solução de Problemas

### O Servidor Não Inicia

Problemas comuns:
- A porta 3306 já está em uso: configure seu servidor para usar uma porta diferente
- Permissões: certifique-se de que os arquivos podem ser lidos/executados
