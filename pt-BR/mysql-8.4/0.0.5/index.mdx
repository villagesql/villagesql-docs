O **VillageSQL Server para MySQL** é um tracking fork extensível e de código aberto, que funciona como um substituto direto para o MySQL.

O VillageSQL Server é construído sobre a base sólida do MySQL, ao mesmo tempo em que introduz um poderoso **Extension Framework**. Esse framework permite que desenvolvedores adicionem capabilities inteiramente novas, como novos tipos de dados e funções, sem esperar por lançamentos upstream ou manter forks complexos.

Esta versão do VillageSQL é compatível com o MySQL 8.4.10.

**<u> A versão 0.0.5 do VillageSQL Server está em DESENVOLVIMENTO. POR FAVOR, NÃO USE EM PRODUÇÃO</u>**
