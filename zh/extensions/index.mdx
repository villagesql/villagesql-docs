VillageSQL 随附内置扩展，并支持由 VillageSQL 团队构建和维护的官方扩展。使用 `INSTALL EXTENSION` 安装任意扩展。

## 内置扩展

<Warning>
  从源码构建时，仅包含 `vsql_complex` 和 `vsql_simple`。以下官方扩展在独立仓库中维护。
</Warning>

### vsql_complex

**状态：** 随 VillageSQL 内置
**仓库：** [villagesql-server/villagesql/examples/vsql-complex](https://github.com/villagesql/villagesql-server/tree/main/villagesql/examples/vsql-complex)

复数数据类型，支持电气工程、信号处理和科学计算的完整算术运算。

**特性：**
- `COMPLEX` 列类型用于存储复数（a + bi）
- 字符串字面量格式：`'(real,imaginary)'`（例如 `'(3.0,4.0)'`）
- 算术：加、减、乘、除
- 工具函数：`complex_real()`、`complex_imag()`、`complex_abs()`、`complex_conjugate()`
- 二进制存储：16 字节（2 个双精度浮点数）

**安装：**
```sql
INSTALL EXTENSION vsql_complex;
```

**示例：**
```sql
CREATE TABLE signals (
    id INT PRIMARY KEY,
    impedance COMPLEX,
    frequency_response COMPLEX
);

INSERT INTO signals VALUES
    (1, '(50.0,0.0)', '(0.95,0.31)');

SELECT
    complex_abs(impedance) as magnitude,
    complex_add(impedance, frequency_response) as total
FROM signals;
```

## 核心扩展

以下扩展在独立仓库中维护，需单独构建和安装。每个扩展提供独立的安装说明和文档。

### vsql_ai

**仓库：** [villagesql/vsql-ai](https://github.com/villagesql/vsql-ai)

AI 驱动的 SQL 函数。支持 AI 提示和嵌入函数用于 SQL 查询，兼容 Anthropic Claude、Google Gemini、OpenAI 和本地 Ollama。

### vsql_crypto

**仓库：** [villagesql/vsql-crypto](https://github.com/villagesql/vsql-crypto)

加密函数。支持哈希（MD5、SHA-1、SHA-256、SHA-512 等）、HMAC、AES 加密、密码哈希和使用 OpenSSL 生成安全随机数据。

### vsql_cube

**仓库：** [villagesql/vsql-cube](https://github.com/villagesql/vsql-cube)

N 维立方体和点类型。存储和查询最多 100 维的几何立方体和点，包含构造器、访问器、谓词、距离和几何函数。需要 VillageSQL 0.0.4。

### vsql_http

**仓库：** [villagesql/vsql-http](https://github.com/villagesql/vsql-http)

HTTP 客户端函数。从 SQL 查询发起 HTTP 请求（GET、POST、PUT、DELETE、PATCH），支持 JSON 响应、URL 编码和解码。

### vsql_network_address

**仓库：** [villagesql/vsql-network-address](https://github.com/villagesql/vsql-network-address)

网络地址类型。支持 IPv4、IPv6 和 MAC 地址数据类型，包含验证、比较和网络操作。

### vsql_uuid

**仓库：** [villagesql/vsql-uuid](https://github.com/villagesql/vsql-uuid)

UUID 数据类型和函数。在 SQL 中原生生成和操作 UUID（v1、v3、v4、v5、v6、v7），提供原生 16 字节 UUID 类型、时间戳提取和比较。

## 社区扩展

社区扩展由第三方使用 VillageSQL 扩展框架（VEF）构建，不由 VillageSQL 团队维护。

### vsql_statistics

**仓库：** [ronaldbradford/vsql-statistics](https://github.com/ronaldbradford/vsql-statistics)
**维护者：** [Ronald Bradford](https://github.com/ronaldbradford)

数据分析的统计聚合函数。支持 IQR、四分位数、中位数和 Tukey 箱线图异常值检测 —— `STATS_IQR`、`STATS_Q1`、`STATS_Q3`、`STATS_MEDIAN`、`STATS_IQR_LOWER_FENCE`、`STATS_IQR_UPPER_FENCE`。需要 VillageSQL 0.0.4。

### prometheus_exporter

**仓库：** [ProxySQL/vsql-prometheus-exporter](https://github.com/ProxySQL/vsql-prometheus-exporter)
**维护者：** [ProxySQL](https://github.com/ProxySQL)

嵌入式 Prometheus 指标导出器。以 Prometheus 曝露格式在 HTTP `/metrics` 端点暴露 MySQL 服务器指标 —— 全局状态、全局变量、InnoDB 指标和复制状态。
