**VillageSQL Server for MySQL** 是一个开源、支持扩展功能的跟踪分叉，可作为 MySQL 的即插即用替代品。

VillageSQL Server 在 MySQL 的坚实基础上引入了强大的 **扩展框架**。该框架允许开发者无需等待上游版本或维护复杂分叉，即可添加全新的功能 — 例如新的数据类型和函数 —。

VillageSQL 的此版本与 MySQL 8.4.9 兼容。

**<u> VillageSQL Server 版本 0.0.4 处于开发中。请勿在生产环境中使用</u>**
