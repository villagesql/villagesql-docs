**VillageSQL Server for MySQL** 是一款开源、可扩展的跟踪分支，可以直接替代 MySQL。

VillageSQL Server 建立在 MySQL 坚实的基础之上，同时引入了一个强大的 **扩展框架**。该框架允许开发人员添加全新的功能——例如新的数据类型和函数——而无需等待上游版本发布或维护复杂的分支。

此版本的 VillageSQL 与 MySQL 8.4.10 兼容。

**<u>VillageSQL Server 版本 0.0.5 处于开发阶段。请勿在生产环境中使用</u>**
