# VillageSQL for MySQL 8.0

Support for MySQL 8.0 is currently in development.

## What to Expect

VillageSQL for MySQL 8.0 will provide:

- Drop-in replacement for MySQL 8.0
- Extension framework support
- All core VillageSQL features

## Current Status

We're actively working on MySQL 8.0 support. In the meantime, you can:

- Use [MySQL 8.4](/mysql-8.4/0.0.1/index) which is currently available
- Join our [Discord](https://discord.gg/KSr6whd3Fr) for updates
- Follow development on [GitHub](https://github.com/villagesql/villagesql-server)

## Stay Updated
